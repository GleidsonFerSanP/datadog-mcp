# Changelog

All notable changes to the "Datadog Observability" extension will be documented in this file.

## [1.0.0] — 2025-01-15

### Added
- Initial release with 65+ Datadog observability tools via MCP
- **Logs**: search, aggregate, tail, patterns, indexes, pipelines
- **Metrics**: query, search, metadata, active metrics, tags, scalar data
- **Monitors**: list, get, search, create, update, delete, mute, unmute, triggered
- **Dashboards**: list, get, create, update, delete, auto-generate observability dashboards
- **Events**: list, get, search, post
- **Incidents**: list, get, create, update, search, todos management
- **APM/Traces**: search spans, aggregate, list services, service health summary
- **Hosts**: list, search, totals, mute, unmute
- **SLOs**: list, get, history, create, delete
- **Synthetics**: list, get, results, trigger, create API tests
- **RUM**: search events, aggregate analytics
- **Security**: list, search, get SIEM signals
- **Downtimes**: list, get, create, cancel
- VS Code extension with credential management via SecretStorage
- Status bar indicator for connection status
- Connection testing command
- Copilot Chat instructions for optimal tool usage
- Support for all 6 Datadog sites (US1, US3, US5, EU, AP1, Gov)
